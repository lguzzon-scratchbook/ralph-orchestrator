# Quick Start

Get Ralph up and running in 5 minutes!

## 1. Initialize a Project

```bash
# Navigate to your project directory
cd your-project

# Traditional mode (simple loop, no hats)
ralph init --backend claude

# Or with a preset workflow
ralph init --preset tdd-red-green

# See all presets
ralph init --list-presets
```

This creates `ralph.yml` in your current directory.

## 2. Define Your Task

Create a `PROMPT.md` file with your task:

```markdown
# Task: Create a Todo List CLI

Build a Python command-line todo list with:
- Add tasks
- List tasks
- Mark tasks complete
- Save to JSON file

Include error handling and a help command.
```

Or pass the prompt inline:

```bash
ralph run -p "Add input validation to the user API endpoints"
```

## 3. Run Ralph

```bash
# TUI mode (default) — real-time terminal UI
ralph run

# With inline prompt
ralph run -p "Implement login endpoint with JWT"

# Headless mode (no TUI)
ralph run --no-tui

# Dry run (show what would execute)
ralph run --dry-run
```

## 4. Monitor Progress

Ralph runs in a TUI by default showing:

- Current iteration number
- Elapsed time
- Active hat (if using hat-based mode)
- Recent agent output

Press `q` to quit the TUI at any time.

## What Happens?

Ralph iterates until one of these conditions:

- `LOOP_COMPLETE` is output (success)
- Maximum iterations reached (default: 100)
- Maximum runtime exceeded (default: 4 hours)
- You quit the TUI

## Basic Configuration

### Minimal Configuration (Traditional Mode)

```yaml
# ralph.yml
cli:
  backend: "claude"

event_loop:
  completion_promise: "LOOP_COMPLETE"
  max_iterations: 100
```

### With Presets (Hat-Based Mode)

```yaml
# ralph.yml — generated by ralph init --preset tdd-red-green
cli:
  backend: "claude"

event_loop:
  completion_promise: "LOOP_COMPLETE"
  max_iterations: 100
  starting_event: "task.start"

# Hats are loaded from the preset
```

## Command-Line Options

```bash
# Limit iterations
ralph run --max-iterations 50

# Use different config file
ralph run -c custom-ralph.yml

# Resume interrupted session
ralph run --continue

# Quiet mode for CI
ralph run -q
```
